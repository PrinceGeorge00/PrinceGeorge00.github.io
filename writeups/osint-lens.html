Overview

Darkseid is a Python-based OSINT tool that extracts image metadata, runs reverse-image search, plots GPS data, and auto-generates PDF reports.

🛠 Features & Workflow

  • ✔ Extracts EXIF metadata from images
  • ✔ Uses Google reverse image search via headless browser
  • ✔ If GPS data present: plots coordinates on a map
  • ✔ Generates a PDF summary with findings

Tools used: Python, Pillow, Selenium, Folium, ReportLab

Installation & Usage

git clone https://github.com/PrinceGeorge00/Darkseid
cd Darkseid
pip install -r requirements.txt
python osint_lens.py --image input.jpg --output report.pdf
